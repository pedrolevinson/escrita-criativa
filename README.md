# A Escrita Criativa

Site do curso on-line **A Escrita Criativa** — uma mentoria em processos de criação com o escritor e roteirista **Bruno Levinson**.

Seis encontros ao vivo, turma limitada a 10 alunos, percorrendo diferentes formas de escrita: projetos culturais, roteiros de TV e documentários, poesia, teatro, letras de música, crônicas, biografias e romances.

## Sobre o site

- Página única (single-page), responsiva, com tema claro/escuro.
- Tudo em um só arquivo: [`index.html`](index.html) — HTML, CSS e imagens embutidas (data URIs). Não depende de nada externo.
- `escrita-criativa.html` é a versão usada como artifact/preview (mesmo conteúdo, sem o wrapper `<html>`).

## Como visualizar

Basta abrir o `index.html` no navegador.

## Como publicar

Por ser um arquivo estático único, pode ser hospedado em qualquer serviço:

- **GitHub Pages:** Settings → Pages → Branch `main` / raiz.
- **Vercel / Netlify:** importar o repositório (build vazio, output = raiz).
